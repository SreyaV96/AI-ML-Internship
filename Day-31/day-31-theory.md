# 1. What is subplot? 
A subplot is a way to display multiple plots within a single figure window. It divides the figure into a grid (rows × columns) and places different graphs in each section.

---
# 2. Why subplots are important? 
Subplots allow:
 - Easy comparison of multiple datasets
 - Better visual organization
 - Efficient use of space in one figure

---
# 3. What is figure size? 
Figure size defines the width and height of the plot (in inches).
It controls how large or small the graph appears.

---
# 4. What is legend? 
A legend explains what each line or dataset represents in a graph.
It helps identify multiple plotted elements.

---

# 5. What is histogram? 
A histogram is a graph that shows the distribution of data by grouping values into bins.
