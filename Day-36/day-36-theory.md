# 1. What is train data? 
Train data is the portion of a dataset used to teach a machine learning model. The model learns patterns and relationships from this data.
# 2. What is test data? 
Test data is the portion of a dataset used to evaluate the model after training. It helps measure how well the model performs on unseen data.
# 3. What is train-test split? 
- Train-test split is the process of dividing a dataset into two parts:
   - Training Data (used for learning)
   - Testing Data (used for evaluation)
- A common split is:
   - 80% Training Data
   - 20% Testing Data
# 4. Why model evaluation is important? 
- Model evaluation is important because it helps:
    - Measure model performance
    - Detect errors
    - Check prediction quality
    - Ensure the model works well on new data
# 5. What is accuracy? 
- Accuracy is the percentage of correct predictions made by a model.

- Formula:

    - Accuracy= Correct Predictions/Total Predictions*100
