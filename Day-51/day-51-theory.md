# 1. What is NLP? 
Natural Language Processing (NLP) is a branch of Artificial Intelligence (AI) that enables computers to understand, interpret, process, and generate human language such as English, Hindi, or Malayalam. It combines computer science, linguistics, and machine learning to allow humans to communicate with computers using natural language.

Example: ChatGPT, Google Translate, and Siri use NLP.
# 2. Why do we need NLP? 
NLP is needed because humans communicate using natural language, while computers understand numbers and programming languages. NLP bridges this gap.

### Benefits of NLP
- Enables communication between humans and computers.
- Automates text analysis.
- Saves time by processing large amounts of text.
- Improves customer service through chatbots.
- Supports language translation and speech recognition.
# 3. Explain three NLP applications. 
### 1. Chatbots
Chatbots answer customer questions automatically using NLP.

Example: ChatGPT, customer support bots.

### 2. Language Translation
NLP translates text from one language to another.

Example: Google Translate.

### 3. Sentiment Analysis
NLP identifies whether a review or comment is positive, negative, or neutral.

Example: Analyzing customer reviews for products.
# 4. Difference between NLP and Computer Vision. 

| NLP | Computer Vision |
|------|-----------------|
| Works with text and language. | Works with images and videos. |
| Input is text or speech. | Input is images or videos. |
| Used for translation, chatbots, and text analysis. | Used for face recognition, object detection, and image classification. |
| Example: ChatGPT | Example: Face Unlock on smartphones |

# 5. What is text data? 
Text data is any information stored in written form using words, sentences, or characters. It can be structured or unstructured.

### Examples
- Emails
- SMS messages
- Social media posts
- Product reviews
- News articles
