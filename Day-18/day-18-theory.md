# 1. What is NumPy? 
- NumPy (Numerical Python) is a core Python library for numerical computing. 
- It provides:
 - Multidimensional arrays (ndarray)
 - Mathematical functions
 - Linear algebra, statistics, and more
 - fast data processing


 ---
# 2. Why NumPy is faster than lists? 
### Problems with Python lists: 
 - Slow for large data 
 - Cannot perform vector operations easily 
### NumPy advantages: 
 - Faster (optimized in C) 
 - Less memory usage 
 - Supports mathematical operations directly


 ---
# 3. What is array?
An array is a ordered collection of elements of the same type stored in contiguous memory 


---
# 4. What is shape and dimension? 
- **Shape**: The number of elements along each axis (e.g., (3,4) means 3 rows and 4 columns).

- **Dimension (ndim)**: The number of axes (e.g., 1D, 2D, 3D).


---
# 5. What is reshaping? 
Changing the structure of an array without changing data.

Example:[1,2,3,4] → [[1,2],[3,4]]
