# 1. What is Matplotlib? 
Matplotlib is a Python library used to create graphs and visualizations like line charts, bar charts, pie charts, etc.

---
# 2. Why visualization is important? 
 - Makes data easy to understand
 - Helps identify patterns and trends
 - Improves decision making
 - Converts complex data into visual form

---

# 3. What is plot? 
- A graphical representation of data points.
- Example: A line plot showing sales growth over months.

---
# 4. Difference between X-axis and Y-axis 
- X-axis: Horizontal axis, usually represents independent variables (e.g., time).
- Y-axis: Vertical axis, usually represents dependent variables (e.g., sales).

---

# 5. Why graphs are important in AI? 
- Help in data analysis (detect patterns and anomalies)
- Visualizes model performance (accuracy, loss curves).
- Helps in feature analysis and data distribution checks.
- Makes results interpretable for humans.
