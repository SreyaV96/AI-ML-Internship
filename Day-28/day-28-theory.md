# 1. What is missing data? 
Missing data refers to the absence of values in a dataset. It can occur due to errors in data collection, incomplete surveys, or system issues.

---

# 2. Why data cleaning is important? 
Data cleaning is important because:
- Ensures accuracy of analysis
- Removes errors and inconsistencies
- Handles missing and duplicate data
- Improves model performance

---
# 3. What is isnull()? 
isnull() detects missing values and returns True/False.

---

# 4. What is dropna()? 
Removes rows or columns containing missing values.

---

# 5. What is fillna()? 
fillna() replaces missing values with custom values(like text, mean, or median)
