# 1. What is a Python library? 
A library is a collection of pre-written code that helps you perform tasks easily. 

---

# 2. Why libraries are important? 
Libraries are important because they:
 - Reduce development time 
 - Reduces code complexity
 - Improve code readability and maintainability
---

# 3. What is math module? 
A built-in Python module that provides mathematical functions like square root, power, trigonometry, and constants like π.

---
# 4. What is random module used for? 
The random module generates random numbers, selects random items, and is used in simulations, testing, and ML experiments.

---
# 5. What is data visualization? 
The process of representing data graphically (charts, plots, graphs) to make patterns and insights easier to understand.
### Libraries:
 - Matplotlib
 - Seaborn
