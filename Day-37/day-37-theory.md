Task 1: Theory (Write Answers) 
# 1. What is regression? 
- Regression is a supervised machine learning technique used to predict a continuous numerical value based on input data.
- Examples:
    - House price prediction 
    - Salary prediction 
    - Temperature prediction 
    - Sales forecasting 

---

# 2. What is classification? 
- Classification is a supervised machine learning technique used to predict a category or class label.
- Example:
    - Spam or Not Spam email
    - Pass or Fail
    - Disease Present or Not Present

---

# 3. Difference between regression and classification 

| Regression                                            | Classification                                                 |
| ----------------------------------------------------- | -------------------------------------------------------------- |
| Predicts numerical values.                            | Predicts categories or labels.                                 |
| Output is continuous.                                 | Output is discrete.                                            |
| Example: House price prediction.                      | Example: Spam email detection.                                 |
| Algorithms: Linear Regression, Polynomial Regression. | Algorithms: Logistic Regression, Decision Tree, Random Forest. |

---

# 4. What is continuous data?
- Continuous data can take any value within a range, including decimal values.
- Examples:
    - Height (170.5 cm)
    - Weight (65.8 kg)
    - Temperature (36.7°C)
    - House Price (₹25,50,000)

---

# 5. What is categorical data? 
- Categorical data represents groups, categories, or labels rather than numerical measurements.
- Examples:
    - Gender (Male, Female)
    - Color (Red, Blue, Green)
    - Department (HR, IT, Sales)
    - Yes/No responses
- Types of categorical data:
    - Nominal: No order (e.g., Color)
    - Ordinal: Has order (e.g., Low, Medium, High)
