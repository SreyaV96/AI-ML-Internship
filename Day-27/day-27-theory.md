# 1. What is CSV file? 
A CSV (Comma-Separated Values) file is a plain text file used to store tabular data. Each row represents a record, and each value is separated by a comma.

---
# 2. Why CSV files are important? 
CSV files are widely used because:
    - They are simple 
    - Easily readable by humans and machines
    - Supported by tools like Excel, Pandas, databases
---
# 3. What is read_csv()? 
In Pandas, read_csv() is a function used to load data from a CSV file into a DataFrame for analysis.

---

# 4. Difference between loc and iloc 
 - loc: Selects rows/columns by labels (names).
 - iloc: Selects rows/columns by integer index positions.

---

# 5. What is data filtering? 
Data filtering means selecting specific rows from a dataset based on conditions.
