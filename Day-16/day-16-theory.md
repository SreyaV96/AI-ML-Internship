# 1.What is data preparation?
Data preparation is the process of collecting, organizing, cleaning, and transforming raw data into a usable format before analysis or machine learning.
# 2.What is data cleaning?
Removing or correcting inaccurate, incomplete, or irrelevant data. Examples: fixing typos, removing duplicates, handling missing values.
# 3.What is missing data?
Missing data refers to values that are absent in a dataset, such as empty cells, None, NULL, or blank values.
# 4.What is normalization?
Normalization is the process of scaling data into a standard range (commonly 0 to 1) so different values can be compared fairly.
# 5.Why data preparation is important?
 - Improves data quality
 - Increases analysis accuracy
 - Removes errors and inconsistencies
 - Saves time in later processing
 - Helps make better decisions
