# 1. What is Naive Bayes? 
Naive Bayes is a supervised machine learning algorithm used for classification. It predicts the class of data based on probability and is commonly used for spam detection, sentiment analysis, and text classification.

# 2. What is Probability? 
Probability is the measure of how likely an event is to occur. Its value ranges from 0 to 1.
- 0 → Impossible event
- 1 → Certain event
- Probability= Number of Favorable Outcomes / Total Number of Outcomes
# 3. Which theorem is used in Naive Bayes? 
Naive Bayes is based on **Bayes' Theorem**.

**Formula:**
P(A|B) = (P(B|A) × P(A)) / P(B)
Where:

- \(P(A|B)\) = Probability of A given B (Posterior Probability)
- \(P(B|A)\) = Probability of B given A (Likelihood)
- \(P(A)\) = Probability of A (Prior Probability)
- \(P(B)\) = Probability of B (Marginal Probability)

# 4. Advantages of Naive Bayes? 
- Simple and easy to implement
- Fast training and prediction
- Works well with large datasets
- Performs well for text classification problems
- Requires less training data
- Handles multiple classes efficiently
