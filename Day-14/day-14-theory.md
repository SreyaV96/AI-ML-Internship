# 1. What is JSON? 
- JSON (JavaScript Object Notation) is a data format used for storing and exchanging data.
- It’s text-based, human-readable, and language-independent.
- Commonly used in APIs and configuration files.
---
# 2. Difference between JSON and dictionary 

| JSON                               | Dictionary                              |
|------------------------------------|------------------------------------------|
| Text format used for data exchange | Python data structure                    |
| Keys must be strings               | Keys can be string, number, tuple, etc.  |
| Uses double quotes                 | Uses single or double quotes             |
| Stored in `.json` files            | Used inside Python programs              |
---
# 3. What is CSV file? 
- CSV = Comma Separated Values 
- Used for storing tabular data. 
# 4. What is virtual environment? 
A virtual environment is an isolated Python environment where packages can be installed separately for each project.
### Benefits:
    - Avoid package conflicts
    - Keep project dependencies separate
    - Easy deployment
---
# 5. What is requirements.txt? 
- It is a text file containing Python packages required for a project.
- Install using: pip install -r requirements.txt
