# 1. What is sorting? 
Sorting is the process of arranging data in a specific order, usually ascending (small to large) or descending (large to small) based on one or more columns.

---
# 2. Why sorting is important? 
Sorting is important because it:
 - Makes data easier to understand
 - Helps in quick searching and analysis
 - Improves data presentation
 - Supports decision-making

---

# 3. What is grouping? 
- Grouping means organizing data into categories based on common values in a column.
- Example: Group employees by department.
- Purpose:

    - Organizes data into categories.
    - Allows comparisons across groups.
    - Often combined with aggregation for deeper insights.

---
# 4. What is aggregation? 
Aggregation summarizes multiple values into a single representative metric.
Common functions:
 - Sum
 - Average
 - Count
 - Maximum
 - Minimum

---
# 5. What is data analysis? 
Data analysis is the process of examining, cleaning, transforming, and interpreting data to extract useful information and support decisions.
