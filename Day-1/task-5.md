# AI vs ML vs Deep Learning

## Artificial Intelligence (AI)

- Artificial Intelligence is the ability of a computer to mimic human behavior.  
- It is a mechanism to incorporate human intelligence into machines through a set of rules (algorithms).  
- Focuses on three major aspects: learning, reasoning, and self-correction

### Types of AI

- **Narrow AI (Weak AI):** Performs a specific task (e.g., voice assistants)  
- **General AI (Strong AI):** Can perform any intellectual task like humans (still theoretical)  
- **Super AI:** Surpasses human intelligence (hypothetical)  

### Examples
- Voice assistants  
- Self-driving cars  
- Chatbots  

---

## Machine Learning (ML)

- Machine Learning is a subset of AI that enables computers to learn automatically from experience and improve without being explicitly programmed.  
- Learns patterns from data  
- Improves with experience  
- Widely used in prediction and data analysis  

### Types of Machine Learning

- **Supervised Learning:** Learns from labeled data to make predictions  
- **Unsupervised Learning:** Finds hidden patterns in unlabeled data  
- **Reinforcement Learning:** Learns through trial and error using rewards and penalties  

---

## Deep Learning (DL)

- Deep Learning is a subset of Machine Learning that uses neural networks to mimic human brain-like behavior.  
- Works with large datasets  
- Automatically extracts features  
- Provides high accuracy for complex problems  
- Can handle unstructured data (images, audio, text)  
- Requires high computational power  

### Core Architecture of Neural Networks

- **Input Layer →** Receives the data (image, text, etc.)  
- **Hidden Layers →** Process data and learn patterns  
- **Output Layer →** Produces the final result

# Top 5 AI Tools in Industry

## 1. TensorFlow 
- Developed by google
- Open-source library for machine learning and deep learning  
- Used to build, train, and deploy AI models  
- Supports large-scale production systems  

## 2. PyTorch 
- Popular deep learning framework  
- Flexible and widely used in research and industry  
- Ideal for building neural network models  

## 3. Hugging Face
- Platform for pretrained models and NLP tasks  
- Used for fine-tuning and deploying large language models  
- Widely used in modern AI applications  

## 4. Microsoft Azure AI 
- Cloud-based AI platform  
- Provides tools for building, training, and deploying models  
- Includes services for vision, NLP, and analytics  

## 5. Databricks
- Unified platform for data engineering, data science, and ML  
- Supports big data processing and ML pipelines  
- Used for end-to-end AI workflow and deployment
