# 1. Define a chatbot.
A chatbot is a software application that interacts with users through text or voice conversations. It is designed to answer questions, provide information, perform tasks, and assist users automatically without requiring human intervention.
# 2. Explain chatbot architecture. 
```text
            User
              │
              ▼
        User Message
              │
              ▼
Natural Language Processing (NLP)
              │
              ▼
     Intent Recognition
              │
              ▼
 Decision & Response Generator
              │
              ▼
      Response to User
```

The main components are:

- User – The person who interacts with the chatbot.
- User Message – The text or voice input provided by the user.
- Natural Language Processing (NLP) – Analyzes and understands the user's message by identifying its meaning.
- Intent Recognition – Determines the user's intention, such as checking an account balance, tracking an order, or resetting a password.
- Decision & Response Generator – Chooses or generates the most appropriate response based on the recognized intent and available information.
- Response to User – The chatbot sends the final reply back to the user.

# 3. What is NLP in chatbots? 
Natural Language Processing (NLP) is a branch of Artificial Intelligence that enables chatbots to understand, interpret, and respond to human language. NLP helps chatbots identify user intent, recognize entities, and provide meaningful responses.

# 4. Differentiate Rule-Based and AI-Based chatbots. 
| Rule-Based | AI-Based |
|------------|----------|
| Uses predefined rules | Learns from data |
| Limited responses | Flexible responses |
| No learning | Learns continuously |
| Easy to build | More complex |
| Suitable for FAQs | Suitable for smart assistants |
