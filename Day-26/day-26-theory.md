# 1. What is Pandas? 
Pandas is a powerful open-source Python library used for data manipulation and analysis. It provides flexible data structures like Series and DataFrame to handle structured data efficiently.

---

# 2. Why Pandas is important? 
Pandas is important because it:
- simplifies handling structured data
- Supports data analysis and preprocessing
- Provides tools for cleaning, filtering, and transforming data
- Makes data handling similar to working with spreadsheets (Excel-like)
- Works well with other libraries like NumPy and Matplotlib

---

# 3. What is Series? 
A Series is a one-dimensional labeled array that can hold data of any type (integers, strings, floats, etc.). It is Similar to NumPy 1D array. 

---
# 4. What is DataFrame? 
A DataFrame is a two-dimensional labeled data structure with rows and columns, similar to a spreadsheet.

---

# 5. Difference between NumPy and Pandas 

| Feature     | NumPy               | Pandas                  |
| ----------- | ------------------- | ----------------------- |
| Structure   | Arrays              | Series & DataFrame      |
| Data Type   | Homogeneous         | Heterogeneous           |
| Purpose     | Numerical computing | Data analysis           |
| Labels      | No labels           | Has row & column labels |
| Ease of use | Lower-level         | More user-friendly      |
