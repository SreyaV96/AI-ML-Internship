# 1. What are aggregation functions? 
- Aggregation functions are functions that perform a calculation on multiple values and return a single result.
- In NumPy, common aggregation functions include:

    - np.sum() → total sum
    - np.mean() → average
    - np.median() → middle value
    - np.std() → standard deviation
    - np.min() → smallest value
    - np.max() → largest value

---
# 2. What is mean? 
- Mean is the average value of data. 
- Formula: 
    Mean = Sum of values / Number of values 
---
# 3. What is median? 
Median is the middle value after sorting the data.

---

# 4. What is standard deviation?
- Standard deviation measures how spread out the values are from the mean.
    - Small standard deviation → values are close to mean
    - Large standard deviation → values are spread out

# 5. What is axis? 
- Axis specifies the direction along which an operation is performed in a NumPy array.
    - axis=0 → column-wise 
    - axis=1 → row-wise
